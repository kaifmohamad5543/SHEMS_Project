# Smart Home Energy Management System (SHEMS)

## Overview
The Smart Home Energy Management System (SHEMS) is a Java-based application developed using Object-Oriented Programming principles, MVC architecture, and software design patterns. The system helps manage household appliances, monitor energy consumption, apply dynamic pricing strategies, schedule appliance operations, and provide real-time notifications within a smart-home environment.

---

## Features

- Appliance Management
- Energy Usage Monitoring
- Dynamic Pricing Strategies
- Device Scheduling
- Malfunction Alert Notifications
- MVC Architecture Implementation
- Object-Oriented Programming Concepts
- Design Pattern Integration
- SHA-256 Password Hashing
- AES Encryption and Decryption
- Functional and Manual Testing

## Design Patterns Implemented

### Singleton Pattern
Used for centralised Energy Management System control.

### Factory Pattern
Used for dynamic appliance object creation.

### Strategy Pattern
Used for multiple pricing strategy implementations.

### Observer Pattern
Used for notification and alert services.
